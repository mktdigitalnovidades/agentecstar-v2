== Description ==

AgentecStar is a modern, one-page WordPress theme designed specifically for AI automation and technology services. The theme features:

* Beautiful gradient backgrounds with brand colors
* Responsive design that works on all devices
* Animated chat interface demonstration
* Smooth scrolling navigation
* Customizable hero section with call-to-action buttons
* WordPress Customizer integration
* WhatsApp integration for direct contact
* Performance optimized
* SEO friendly
* Translation ready

Perfect for:
- AI and automation service providers
- Tech consulting companies
- Digital agencies
- SaaS companies
- Chatbot services
- WhatsApp automation services

== Installation ==

1. Upload the theme folder to your /wp-content/themes/ directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Go to Appearance > Customize to configure the theme settings
4. Add your WhatsApp number and customize the text content
5. Upload your logo through Appearance > Customize > Site Identity

== Customization ==

The theme includes extensive customization options through the WordPress Customizer:

* Hero Section: Customize badge text, brand name, subtitle, description, and CTA buttons
* Contact Settings: Set WhatsApp link and contact information
* Services Section: Configure services title and subtitle
* Colors: The theme uses a predefined color scheme optimized for AI/tech branding
